# Context Engine

A tool that parses codebases into structured representations 
using Abstract Syntax Trees — mapping functions, dependencies, 
and call relationships.

Built as the foundation for a context-aware AI pair programmer 
that understands code structurally, not just as text.

## Usage
python context.py <path_to_folder>

## Current Features
- Scans Python codebases and extracts all function definitions

## Roadmap
- Call graph generation
- Dependency mapping
- AI integration on top of structured context
